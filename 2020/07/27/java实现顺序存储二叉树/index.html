顺序存储二叉树的特点：（这里的n指的是元素当前的索引值 ）
1.顺序二叉树通常只考虑完全二叉树
2.第n个元素的左子节点为： 2*n + 1
3.第n个元素的右子节点为： 2 *n + 2 (因为它是右子树所以在左子树的基础上加1)
4.第n个元素的父节点为：(n - 1)/2
