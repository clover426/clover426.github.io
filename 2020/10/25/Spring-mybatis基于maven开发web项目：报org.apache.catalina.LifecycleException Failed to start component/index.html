我先把错误的信息给大家给出来：

我配置的pom.xml

导致出现这个问题的原因是：

注意：我们本项目使用的servlet-api是3.0，项目打成war包后就有了，但是本地的tomcat也有一个2.5，这时这两个jar包就冲突了，不知道使用哪个了，才会发生这个错误！我们的第一种做法就是将maven中配置的servlet-api与jsp-api的scope设置为provided，即生成的源码不打入以jar包与war包中，就不会出现jar包冲突的情况了！

所以我们现在改掉这两个jar包的scope
